## 🚀 Getting Started

### Install Jupyter

```bash
pip3 install jupyter bash_kernel
python3 -m bash_kernel.install
```

### Launch

```bash
cd ~/Repo/bash-pi-openclaw-education/notebooks/
jupyter notebook
```

### Run Cells

- Click in a cell
- Press `Shift + Enter` to run
- Output appears below

## 💡 Tips

**Try variations:**
```bash
# Original
ls -la

# Try different flags
ls -lh
ls -ltr
```

**Chain commands:**
```bash
# Build up gradually
ls
ls | wc -l
ls | grep ".md" | wc -l
```

**Save your experiments:**
- File → Save As
- Create your own notebooks
- Document what you learned
